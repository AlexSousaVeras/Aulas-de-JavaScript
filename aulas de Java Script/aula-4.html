Iniciando estudos no DOM

DOM é um Acrônimo para Document Object Model(Modelo de Objetos para documentos)

Devo aprender a fazer e a usar a Árvore DOM, essa árvore cria raizes, e essas reizes se chamam 'window', tudo dentro do JavaScript está denro de um window.

Dentro do window tem vários outros três objetos, por exemplo: location, document e history

location: Fala qual é a localização do WebSite, a URL, qual a página anterior...

document: Esse é o documento atual. Dentro dele fica o documento HTML, e dentro do mesmo fica o 'head' e o 'body'

history: Diz de onde eu vim, para onde eu vou, isso facilita a navegação dento do site.


Como navegar pelos elementos

Métodos para selecionar os elementos:

por Marca: getElementsByName()

por ID: getElementById()

por Nome: getElementsByName()

por Classe: getElementByClassName()

por Seletor: querySeletor() e querySeletorAll(). O querySeletor() é no singular, e o querySeletorAll() é no plural. E esse método é o mais recomedado
