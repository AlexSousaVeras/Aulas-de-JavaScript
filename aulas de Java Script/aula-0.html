Variáveis e tipos primitivos

'=' siginfica 'recebe'

'==' significa 'igual'

'===' siginfica 'igualdade estrita, ou seja, somente retorna true se odd operadores forem do mesmo tipo e valor'

'null' significa nula, sem nada.

'var' significa variável

'let' também significa variável, esse é usado no JavaScript moderno

As variáveis podem começar com letras, $ ou _

Não podem começar com números

É possível usar letras com números

É possível usar acentos e símbolos

Não podem conter espaços

Não podem ser palavras reservadas, como: function, alert, window, var, let...
